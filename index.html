Stuff I've done;

This a short overview of some of the projects I've completed both at University and in my spare time.

Radio Access Technology Selection and Simulation Algorithm

For a UCT course in mobile and wireless networking, I developed and tested an algorithm to allocate simulated mobile users to multiple simulated radio towers. These decisions used fuzzy logic to allocate the users based on the their traffic type, their network preferences, as well as the capabilities of the radio towers. This project strengthened my skills in MATLAB coding, multi-criteria decision making, and data representation.

Machine Learning System to Identify Military Vehicles from SAR Images

Machine learning is what I consider to be my specialty. While the implementation of this feedforward neural network was far from complex, it taught me the most and served as my introduction into this wonderful field. This was the first ML project that I truly completely optimised, reaching accuracy percentages in the high 90s.

Compression and Encryption Algorithms for Resource Constrained Embedded Systems

This group project required myself and a friend to design and implement data compression and encryption on an STM microcontroller. I was made responsible for the compression specifically. While this project seriously tested and strengthened my programming skills in embedded C, my greatest takeaway was the understanding of the importance of efficiency. Getting our algorithms to run on such a resource constrained platform was hard enough, but getting them to run well was an incredibly rewarding challenge.

Autonomous Bird Weigh Station with Remote Access

Myself and 3 other students were tasked with identifying and solving a problem faced by ornithologists at the Fitzpatrick Institute of African Ornithology. Our end product was very successful and was one of the few in the class that actually worked. My subsystem was a database with remote access. This fortified my skills in, to name a few, database design, user interface design, embedded networking, and cyber security. My favourite part of the project was the final pitch to the stakeholders, whom were very pleased with their new toy.
